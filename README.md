## Installation

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd AI-nihongo-kaiwa
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the application**
   ```bash
   # Development mode
   npm run dev

   # Or build first then run
   npm run build
   npm start
   ```

4. **Open browser**
   Open http://localhost:3000 in your browser

4. **Listen to AI response** in Japanese
